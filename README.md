# Overview

EyeSpy is a travel safety platform that helps users quickly assess hotel rooms and short-term rentals the moment they arrive. Using a phone-based room walkthrough, EyeSpy analyzes the space for potential risks such as suspicious devices, weak door or lock security, exposed wiring, missing safety equipment, and unfamiliar network activity. The platform then generates a clear safety summary, provides voice-guided findings, and creates a shareable verified report that can be used for documentation, disputes, or peace of mind while traveling. For the MVP, EyeSpy focuses on combining AI-powered visual analysis, device scanning, and reporting into one smooth experience through a web-based interface.
